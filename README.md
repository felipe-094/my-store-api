# 🏪 My Store API

API REST desarrollada con Node.js y Express para la gestión de productos y práctica de conceptos fundamentales del desarrollo backend.

## ✨ Características

* Gestión de productos mediante endpoints REST.
* Uso de middlewares personalizados.
* Manejo centralizado de errores.
* Configuración de CORS.
* Arquitectura modular.
* Validación de datos mediante schemas.

## 🛠️ Tecnologías utilizadas

* Node.js
* Express.js
* JavaScript
* CORS
* Git
* GitHub

## 🚀 Instalación

Clonar el repositorio:

git clone https://github.com/felipe-094/my-store-api.git

Entrar al proyecto:

cd my-store-api

Instalar dependencias:

npm install

Ejecutar el servidor:

npm start

## 📌 Endpoints

Ejemplo de consulta:

GET /api/v1/products

Este proyecto fue desarrollado como parte de mi proceso de aprendizaje en el desarrollo de APIs con Node.js y Express, fortaleciendo conocimientos en rutas, middlewares y organización del código.
